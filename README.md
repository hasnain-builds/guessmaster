# GuessMaster

A polished, browser-based number guessing game built with HTML, CSS, and vanilla JavaScript. The experience is designed for GitHub Pages and other static hosts, with no backend required.

## Features

- GuessMaster branded logo system with landing, loading, victory, and game-over states
- Three difficulty levels with attempt tracking
- LocalStorage stats, best score, and achievement badges
- Sound toggle, dark and light mode, confetti, and polished UI motion
- Responsive glassmorphism layout with animated gradients and floating particles

## Run Locally

Open [index.html](index.html) directly in a browser, or serve the folder with any static server.

## GitHub Pages

1. Push the repository to GitHub.
2. Enable GitHub Pages in repository settings.
3. Select the root branch and root folder.
4. The game will deploy from [index.html](index.html).

## Legacy CLI

The original Python starter remains in [main.py](main.py) as the gameplay reference.
