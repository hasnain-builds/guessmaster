# 🎯 GuessMaster

**Think. Guess. Win.**

GuessMaster is a modern browser-based Number Guessing Game built using HTML, CSS, and Vanilla JavaScript. Originally developed as a Python CLI project and later transformed into a polished web application with a premium gaming-inspired interface.

## 🚀 Live Demo

Play Now:
https://hasnain-builds.github.io/guessmaster/

---

## ✨ Features

### 🎮 Gameplay

* Random number generation
* Difficulty selection system
* Real-time guess validation
* High / Low hints
* Win & Game Over screens
* Restart game functionality

### 🎯 Difficulty Modes

| Difficulty | Range   | Attempts |
| ---------- | ------- | -------- |
| Easy       | 1 - 50  | 4        |
| Medium     | 1 - 100 | 5        |
| Hard       | 1 - 500 | 8        |
